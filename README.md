# Calcea's Repository  
A collection of various GIS websites.

## Current Weather  
A map displaying current and anticipated weather conditions in the United States.  Depicts real-time data weather radar data across the US, and provides alerts directly from the National Weather Service.  
<https://calcea-j.github.io/weather>

## Photo Map Tour
A map of some restaurants around Baton Rouge, LA that resonate with college students. Allows users to click, and show a photo of the locations with a description.
